# Restaurant Food Ordering App

A responsive and dynamic **Restaurant Food Ordering Web Application** built using **React.js** that allows users to browse menus, view food details, add items to cart, and place orders seamlessly.

---

## 🚀 Tech Stack
- ReactJS (TypeScript)
- Redux
- Tailwind CSS
- Firebase

---

## 🌐 Live Demo
https://zone-restaurant.vercel.app/

---

## 📱 Features
- Browse food menu
- Add to cart
- Remove items from cart
- Checkout system
- Responsive design (Mobile + Desktop)
